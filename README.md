# Meu Portal de Entretenimento Digital

Este projeto foi desenvolvido para a atividade de HTML e CSS.

O site possui três seções principais:

- Filmes e Séries
- Música e Podcasts
- Games e Tecnologia

## Tecnologias utilizadas

- HTML5
- CSS3
- Flexbox

## Recursos utilizados

- CSS Inline
- CSS Interno
- CSS Externo
- Classes
- IDs
- Reset CSS
- Comentários no CSS

## Objetivo

Criar uma página web temática utilizando conceitos básicos de HTML e CSS.
